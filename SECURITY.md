## Security

Text Shot is a local macOS OCR utility. All capture and OCR processing occurs on-device.
The app does not require network access during normal operation.

### Reporting a Vulnerability

Please report security issues privately.

Preferred:
- Use GitHub Security Advisories for this repository.

If private reporting is unavailable:
- Open a GitHub issue with minimal, non-sensitive details and request a private contact channel.

When reporting, include:
- Affected version and macOS version
- Steps to reproduce
- Expected vs actual behavior
- Security impact

### Disclosure

Please do not publicly disclose vulnerabilities until a fix or mitigation is available.
